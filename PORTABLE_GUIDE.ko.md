# 🎒 EasyGo Issue 휴대용 가이드 (Portable Guide)

**EasyGo Issue**는 개발 환경이 갖춰지지 않은 PC에서도 즉시 실행할 수 있도록 설계되었습니다.

## 1. 내장 Node.js 환경
이 패키지에는 **공식 Node.js 실행 파일 (`bin/node.exe`)**이 포함되어 있습니다.
- **무설치**: 별도로 다운로드하거나 설치할 필요가 없습니다.
- **무간섭**: 시스템 레지스트리를 건드리지 않습니다.
- **안전함**: 공식 배포판을 그대로 사용합니다.

## 2. 배포 및 이동 방법
이 프로그램은 폴더 자체로 완결된 구조를 가집니다.

1.  개발 PC에서 `npm run build`를 통해 `dist` 폴더를 생성합니다.
2.  **프로젝트 폴더 전체**를 USB나 공유 폴더로 복사합니다.
3.  대상 PC에서 `start_app.bat`를 실행하면 설치 과정 없이 바로 실행됩니다.

## 3. 데이터 관리 (백업/복구)
모든 데이터는 단 **하나의 파일**에 보관되므로 관리가 매우 간편합니다.

- **데이터 파일 경로**: `server/database.sqlite`

### 💾 백업 방법
1.  프로그램을 종료합니다 (터미널 창 닫기).
2.  `server/database.sqlite` 파일을 복사하여 USB나 클라우드 등 안전한 곳에 저장합니다.

### 🔄 복구 방법
1.  프로그램을 종료합니다.
2.  백업해둔 파일을 `server/` 폴더 안에 덮어씌웁니다 (기존 파일 대체).
3.  다시 프로그램을 실행하면 예전 데이터로 복구됩니다.
